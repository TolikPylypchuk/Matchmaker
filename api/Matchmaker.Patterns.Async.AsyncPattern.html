EqualTo<TInput>(Func<Task<TInput>>)

Returns a pattern which is matched successfully when the input value is equal to the provided value.

public static IAsyncPattern<TInput, TInput> EqualTo<TInput>(Func<Task<TInput>> valueProvider)

Parameters

valueProvider Func<Task<TInput>>

The provider of the value to check for equality.

Returns

IAsyncPattern<TInput, TInput>

A pattern which is matched successfully when the input value is equal to the provided value.

Type Parameters

TInput

The type of the input value of the expression.

Remarks

The valueProvider is not called until this pattern's MatchAsync(TInput) method is called.

The valueProvider will be memoized, so it will be called once, and then its result will be cached. The caching process is not thread-safe, so there is a chance that the valueProvider can be called more than once.

If valueProvider returns null, an InvalidOperationException will be thrown.

Exceptions

ArgumentNullException

valueProvider is null.
