Interface IPattern<TInput>

Represents a pattern to match with in a match expression. This inteface is for internal use and should never be used directly by the users of the library. Instead, users should use IPattern<TInput, TMatchResult>

Namespace: Matchmaker.Patterns
Assembly: Matchmaker.dll
Syntax
public interface IPattern<in TInput>
Type Parameters
Name Description
TInput

The type of the input value of the match expression.
Remarks

This interface exists only for internal use in the Match<TInput, TOutput> and Match<TInput> classes, because they can't know about the types of the transformation results. The public API never uses this interface - it always uses IPattern<TInput, TMatchResult>, so type safety is not compromised.
