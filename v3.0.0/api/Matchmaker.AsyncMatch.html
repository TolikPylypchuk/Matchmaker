ClearCache<TInput>()

Clears the global cache of static AsyncMatch<TInput> instances.

Declaration
public static void ClearCache<TInput>()
Type Parameters
Name Description
TInput

The type of the input value.
Remarks

After calling this method all CreateStatic<TInput>(Action<AsyncMatchBuilder<TInput>>, String, Int32) calls will recalculate match statements for type TInput.

Clearing the cache is thread-safe.

CreateStatic<TInput>(Action<AsyncMatchBuilder<TInput>>, String, Int32)

Creates an asynchronous match statement which is then globally cached.

Declaration
public static AsyncMatch<TInput> CreateStatic<TInput>(Action<AsyncMatchBuilder<TInput>> buildAction, string sourceFilePath = "", int sourceLineNumber = 0)
Parameters
Type Name Description
Action<AsyncMatchBuilder<TInput>> buildAction

The action which builds the match statement.
System.String sourceFilePath

The path to the source file in which this method is called.
System.Int32 sourceLineNumber

The line in the source file where this method is called.
Returns
Type Description
AsyncMatch<TInput>

The globally cached match expression.
Type Parameters
Name Description
TInput

The type of the input value.
Remarks

The key of the cache is the location of this method's call in the source code. This way, a new match statement object will not be allocated every time pattern matching occurs in the same place more than once.

The pattern caching process is thread-safe.
